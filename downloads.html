Dependencies

{{ page.label-begin-minecraft }} MINECRAFT {{ page.label-end }}

The required version of Minecraft to run a build. Builds are compiled using SRG names, which may enable them to run on later minor builds of Minecraft. Please keep in mind this does not guarantee stability unless the Minecraft version matches exactly.

{{ page.label-begin-forge }} FORGE {{ page.label-end }}

The minimum Forge version necessary to run a build.

Additional Files
Chisel Design Packs

Chisel Design Pack by Wreckage

This pack includes over 30 chisel designs. To use these custom designs, drag the image files you want into the mod zip file under: assets/carpentersblocks/textures/blocks/designs/chisel/


Tile Design Packs

Kaevator Wallpaper Tile Design Pack

Since Kaevator's Wallpaper mod was the inspiration behind tiles, I've compiled the original artwork in a format ready to load as custom tile designs. Simply place the image files into the mod zip file under: assets/carpentersblocks/textures/blocks/designs/tile/
